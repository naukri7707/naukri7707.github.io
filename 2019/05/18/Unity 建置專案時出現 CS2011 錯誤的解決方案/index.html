    Unity 建置專案時出現 CS2011 錯誤的解決方案

    2019-05-18

    錯誤訊息

    CS2011
    無法開啟回應檔 ‘file’
    編譯時已指定回應檔，但編譯器找不到檔案而無法加以開啟。

    錯誤原因

    為了改變CMD預設路徑，新增了AutoRun字串 機碼導致編譯時開啟錯誤的資料夾

    解決方法

    刪除 HKEY_CURRENT_USER\Software\Microsoft\Command Processor 下註冊的AutoRun 字串

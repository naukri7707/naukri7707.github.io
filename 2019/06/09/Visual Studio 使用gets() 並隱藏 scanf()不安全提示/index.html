Visual Studio 使用gets() 並隱藏 scanf()不安全提示
  • Visual Studio
2019-06-09

使用 gets()

原因

gets() 在 C++11 時被棄用，並在 C++14 時被刪除，取代方案為 gets_s() 但 OJ 版本不到 C++11 時會沒有 gets_s()能用。

解決方案

找到 Visual Studio 的 stdio.h ，可以使用 VisualStudio 的前往文件找到，並使用令存新檔找到路徑。使用檔案總管找到 stdio.h 並複製兩份到桌面，一份修改一份備用，編輯桌面上的 stdio.h 用搜尋功能搜尋 gets_s 並找到他

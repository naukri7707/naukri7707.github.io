模板

Stack

概述

本組模板為堆疊模板，在已知元素數量上限時使用此模板可以減去動態宣告記憶體的時間，本模板可同時宣告並使用複數個 stack，而只須保證元素總數不超過定義上限即可。

結構

Node

本結構為使用者自定義的資料結構。

Stack

本結構為堆疊結構，使用他來宣告一個新的堆疊。

函式

push()

本函式會新增一筆資料至目標堆疊的頂端。

參數
stk (Stack**)

本參數為目標堆疊。

val (Node)

本參數為存入目標堆疊頂層的值。

回傳值

本函式無回傳值。

pop()

本函式會將目標堆疊頂層的資料取出並回傳該值。

stk (Stack**)

本參數為目標堆疊。

回傳值

本函式回傳目標堆疊頂層元素。

其他

STKSIZ

此巨集用來定義總元素數量的上限。

實例

編譯運行結果

0,2
0,1

注意事項

  • 測試階段，謹慎使用。

概述

本組模板為輸入優化模板，使用此模板取代 scanf() 可以大幅減少從資料流取得數組的時間，本組模板所有函式皆需要透過 readChar() 完成，請在使用時一併引入。

參數

BUFMAX

此巨集用來定義緩衝區大小。

dst

本參數用來接收格式化後的數值。

返回值

若讀取到字元 -1 (EOF) 回傳 0，否則回傳 1。

實例

注意事項

為了加快速度，過濾字元時使用 (ch < '0' (有號為'-'))、判斷數字使用 (ch >= '0') 作為字元判斷依據如果字串中包含不符合條件的字元請自行修改判斷式或採用以下嚴格判斷的方式避免錯誤。

過濾字元

有號

(((ch = *buffer++) < '0' || ch > '9') && ch != '-')

無號

((ch = *buffer++) < '0' || ch > '9')

判斷數字

((ch = *buffer++) >= '0' && ch <= '9')

[模板] 字串取值

  • 字串處理
  • 模板
2019-08-27

模板

getInt()

getUInt()

getLongLong()

getULongLong()

getDouble()

概述

本組模板為字串取值模板，使用此模板取代 sscanf() 可以大幅減少從字串取得數組的時間，此外本模板內建指針偏移，可以節省使用 strlen() 偏移指針的開銷。

參數

dst

本參數用來接收格式化後的數值。

src

本參數為來源字串。

返回值

該函數返回最後一個格式化數字的下一位元指標，如果過濾字元時檢索到空字元 ('\0')，則返回一個空指標。

實例

編譯運行結果

123
20
0
-17

注意事項

為了加快速度，過濾字元時使用 (ch < '0' (有號為'-'))、判斷數字使用 (ch >= '0') 作為字元判斷依據如果字串中包含不符合條件的字元請自行修改判斷式或採用以下嚴格判斷的方式避免錯誤。

過濾字元

有號

(((ch = *buffer++) < '0' || ch > '9') && ch != '-')

無號

((ch = *buffer++) < '0' || ch > '9')

判斷數字

((ch = *buffer++) >= '0' && ch <= '9')

使用 gets()

原因

gets() 在 C++11 時被棄用，並在 C++14 時被刪除，取代方案為 gets_s() 但 OJ 版本不到 C++11 時會沒有 gets_s()能用。

Unity 建置專案時出現 CS2011 錯誤的解決方案
  • Unity

錯誤訊息

CS2011
無法開啟回應檔 ‘file’
編譯時已指定回應檔，但編譯器找不到檔案而無法加以開啟。

錯誤原因

為了改變 CMD 預設路徑，新增了 AutoRun 字串 機碼導致編譯時開啟錯誤的資料夾

解決方法

刪除 HKEY_CURRENT_USER\Software\Microsoft\Command Processor 下註冊的 AutoRun 字串

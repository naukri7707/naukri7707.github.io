Visual Studio 使用gets() 並隱藏 scanf()不安全提示

使用 gets()

原因

gets() 在 C++11 時被棄用，並在 C++14 時被刪除，取代方案為 gets_s() 但 OJ 版本不到 C++11 時會沒有 gets_s()能用。

解決方案

找到 Visual Studio 的 stdio.h ，可以使用 VisualStudio 的前往文件找到，並使用令存新檔找到路徑。使用檔案總管找到 stdio.h 並複製兩份到桌面，一份修改一份備用，編輯桌面上的
stdio.h 用搜尋功能搜尋 gets_s 並找到他

存取修飾詞

public

任何類別皆可存取

protected

只有自身類別或繼承自身的類別可以存取

internal

只有同一組件(不是命名空間)下的類別可以存取

protected internal

符合 protected 或 internal 任一條件即可進行存取 (自身類別 / 繼承自身的類別 / 同一組件)

private

只有自身類別可以存取

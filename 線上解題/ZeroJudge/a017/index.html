a017: 五則運算

  • 後序式
2019-08-03

內容

計算五則運算式的結果，包含加、減、乘、除、餘

輸入

輸入一個字串，其中包含運算元及運算子，為了方便讀取，所有的運算子及運算元均以空格區隔。

運算元為 0 ~231 -1 的整數

運算子則包含 + - * / % 及 ( )

運算時請注意先乘除後加減及() 優先運算的計算規則

2 + 3 _ 4
2 _ ( 3 + 4 ) * 5

輸出

輸出結果。為了避免小數點誤差，所有的運算過程都不會產生小數點，可以放心使用整數進行運算

14
70

解題思路

先把輸入的中序式轉成後序式，再解析後序式轉成整數。

完整程式碼

AC (1ms, 96KB)

a694: 吞食天地二

2019-08-13

內容

好餓歐歐歐歐

有 n * n 個食物在你面前排成一個方陣

每個食物有它的飽足度

你想知道把其中一塊通通吃掉會獲得多少飽足度

輸入

多組測資以 EOF 結束

每組測資開始有兩個正整數 n,m (n<=500, m<=100000)

接下來 n 行有 n 個不超過 100 的正整數依序代表每個食物的飽足度

接下來 m 行每行有四個數字 x1,y1,x2,y2 (1 <= x1 <= x2 <= n, 1 <= y1 <= y2 <= n)

代表你想要吃掉食物的範圍

3 3
1 2 3
4 5 6
7 8 9
1 1 3 3
1 1 1 3
1 1 3 1

輸出

對每組測資輸出 m 行，代表總飽足度

45
6
12

解題思路

吞食天地 的加強版，從一維變成二維，但解題邏輯一樣是建立總飽足度陣列之後用數學去取得目標區間。

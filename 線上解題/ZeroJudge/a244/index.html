a244: 新手訓練 ~ for + if

內容

內容就是~

希望學到for迴圈和剛開始coding的學弟好好加油!!!!

輸入

第一行有一個正整數N，

代表接下來有N行每行有三個正整數 a , b , c

( 1 <= b , c <= 2147483647 )

( 1 <= a <= 4 )

4
1 2 3
2 2 3
3 2 3
4 2 3

輸出

如果 a = 1 請輸出 b+c

如果 a = 2 請輸出 b-c

如果 a = 3 請輸出 b*c

如果 a = 4 請輸出 b/c

結果請用整數輸出

5
-1
6
0

簡單的三元運算子判斷。
